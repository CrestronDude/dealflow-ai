# DealFlow AI — Viral Loops

## Loop 1: Deal Share
- User analyzes deal → gets shareable PDF report
- PDF has "Analyzed by DealFlow AI" watermark + QR code
- Partners who receive report sign up → referral credit

## Loop 2: Referral Program
- Share link → friend gets 1 free extra analysis
- Referrer gets 1 free extra analysis per signup
- Tracked via Zapier + Supabase

## Loop 3: Community Embed
- "Post this deal to your investor group" button
- Auto-formatted for Facebook Groups, Slack, Discord
- Includes deal score card image (og:image generated)

## Loop 4: Weekly Digest
- Weekly email: "Your portfolio this week"
- Shareable "I made $X this month from rentals" card
- Instagram/Twitter-ready image via API
